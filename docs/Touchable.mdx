# Touchable

Touchable renders a TouchableOpacity or a TouchableHighlight

## Basic usage

<Playground>
  <Touchable
    bg="cornflowerblue"
    p={3}
    m={2}
    justifyContent="center"
    alignItems="center"
  >
    <Text color="white">Touchable</Text>
  </Touchable>
</Playground>

## Props

- Any react-native-kondo `<Box />` props
- Any React Native `<TouchableHighlightProps />`, `<TouchableNativeFeedbackProps />`, `<TouchableOpacityProps />`, `<TouchableWithoutFeedback />` props
